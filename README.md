# 🌐 Journey to CCNA & Beyond

Welcome to my learning repository! I am an aspiring Network Engineer documenting my journey through the Cisco Certified Network Associate (CCNA) curriculum. This repository serves as a **"Build-in-Public"** portfolio to track my technical progress and develop analytical documentation skills for my future **EPQ (Extended Project Qualification)** at A-Level.

My goal is to blend technical engineering discipline with reflective thinking—showing not just *what* I build, but *how* I solve problems.

## 🚀 The Game Plan & Structure
I am systematically documenting every section of the course:

* `weekly-synthesis/` - Polished, weekly summaries detailing engineering insights and soft-skill reflections.
* `labs/` - Cisco Packet Tracer network topologies (`.pkt`) and active configuration files.
